# Enrollee-Angular-CRUD

## Getting Started

1. Clone this repository.

    `git clone https://github.com/ashishshah1995/Enrollee-Angular-CRUD.git`

    You could also start with an empty project but this repo contains everything you need to get started quickly: theming, configured routing and a menu.

2. Install the needed npm packages:

   `npm install`

3. Run Deno service
Go to Server folder and Run `deno run --allow-net server.ts --port=8081`

4. Development server
Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`.
